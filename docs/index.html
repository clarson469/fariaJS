faria.random.choose( array )

Selects a random item from a given array

Arguments
array (Array) The array from which an item will be chosen
Returns

faria.random.choose(['a', 'b', 'c']);
  // => 'a' or 'b' or 'c'

randInt()

faria.random.randInt( a, [b] )

Generates a random integer from within the specified range

Arguments
a (Number) If this is the only argument provided, the range of numbers from which the function generates an integer is 0 - a. Otherwise, it is the first of the range bounds

[b] (Number) The upper bounds of the range of numbers from which an integer is generated
Returns
(Number) A random integer 
faria.random.randInt(2);
// => 0 or 1

faria.random.randInt(2, 4);
// => 2 or 3

getBetween()

faria.string.getBetween( string, a, [b, keepSymbols=true])

Returns a list of substrings from the given string defined by given surrounding characters

Arguments
string (String) The string to be searched

a (String) If b is not given, this serves as both of the surrounding characters used to identify the desired substrings, otherwise it is the first of those characters

b (String) If passed, the closing character used to identify the desired substrings

keepSymbols (Bool) If true, the substrings returned include their identifying characters
Returns
(Array) An array of all the substrings identified by the given characters. If no substrings can be found, an empty array is returned 
faria.string.getBetween('get %this%', '%');
// => [ '%this%' ]

faria.string.getBetween('get %this% and %that%', '%', null, false);
// => [ 'this', 'that' ]

faria.string.getBetween('get {this} and {that}', '{', '}');
// => [ '{this}', '{that}' ]

formatText()

faria.string.formatText( string, args )

Replaces array keys embeded in a string with corresponding substrings

Arguments
string (String) The string to be formatted, containing keys to be replaced contained within curly brackets: "{}"

args (Array | Object | String | Number) The substrings to be inserted into the string, organised according to what key they should replace. If a string or an number is given, the function converts that to a single-item array, and so it should be denoted accordingly in the string argument
Returns
(String) The same string that was passed, but with all denoted keys replaced with their respective substrings 
faria.string.formatText('the {0} where it {1}', ['room', 'happens']);
// => 'the room where it happens'

faria.string.formatText('My name is {name}, I am {age} years old.', {name:'Joe', age:31});
// => 'My name is Joe, I am 31 years old.'

faria.string.formatText('{0} {0} {0}nd', 'la');
// => 'la la land'

faria.data

getUnique()

faria.data.getUnique( array )

Returns the given array with all duplicated items removed - a set.
The function does not confuse between like values: '1' and 1 are treated as being different, nor does it confuse like types: {key:'a'} and {key:'b'} are also treated as being different.
(Note that the function may take a while if given a very large array containing a number of complex objects)

Arguments
array (Array) The array whose duplicates will be removed
Returns
(Array) The given array, with all duplicates removed 
faria.data.getUnique([1,1,2,3,2,4]);
// => [ 1, 2, 3, 4 ]

faria.data.getUnique(['1', 1, {dup: 1}, {dup: 1}, {dup: 2}, ['a', 'b'], ['a', 'b']]);
// => [ '1', 1, {dup:1}, {dup:2}, ['a', 'b'] ]

getChildren()

faria.data.getChildren( object, [parentString=null, excl=[]] )

Returns an array of all the items at a given position, or depth, of an an object

Arguments
object (Object) Usually an object-literal (although a proper, constructed object-function could be used in certain cases, though it would not be advised), this is the object to be searched

parentString (String) A string representation of the path through the object to the position required. If not given, the function uses the top level of the object
(Note that the function returns items of an object, so parentString should always point to a position in the object that still itself contains keys, and not to a deepest position in the object)

excl (Array) An array of the names of any keys whose items you do not want to be included in the returned array
Returns
(Array) An array of the selected items 
faria.data.getChildren({one:1, two:2, three:3});
// => [ 1, 2, 3 ]

faria.data.getChildren({
  mod:{
    mod_2:{
      hello:'world'
    }
  }
}, 'mod.mod_2');
// => [ 'world' ]

faria.data.getChildren({
  mod:{
    mod_2:{
      yes:'hello',
      no:'goodbye'
    }
  }
}, 'mod.mod_2', ['no']);
// => [ 'hello' ]

findInObjList()

faria.data.findInObjList( array, key, value )

Searches an array of objects for an object whose key identifies a specific value. Useful when you have a number of objects in an array, and know that that all have the same structure, but don't know exactly what they all look like.
(Note that this function does rely on the objects having some kind of unique key or property that you know beforehand)

Arguments
array (Array) The array of objects to search

key (String) The key by which to identify the desired object

value (*) The value of the desired object's key
Returns
(Object) The desired object 
faria.data.findInObjList([
  {key: 1},
  {key: 2}
], 1);
// => { key: 1 }

assignNodeValue()

faria.data.assignNodeValue( object, nodeString, value )

Navigates to a specified position in the given object, and assigns that position a value.
Useful for when you need to assign a value to a position in an object, but cannot know exactly what that position will be
(Note that if the specified position does not currently exist, the function will create it before assigning it a value)

Arguments
object (Object) The object to change

nodeString (String) A string representing the path to the desired position in the object. Each layer of the object must be denoted by a full stop: "."

value (*) The value to assign 
var obj = {
  mod:{
    mod_2:{
      target:null
    }
  }
};

faria.data.assignNodeValue(obj, 'mod.mod_2.target', 'hello world!');

obj;
// => { mod: { mod_2: { target: 'hello world' } } }

randomiseStructure()

faria.data.randomiseStructure( object )

If the given object, at any position, contains an array, this function will replace that array with a randomly chosen value from the same array
(Note that, were you you to look at the source code for this function, you would see that it actually accepts two arguments, however, the second argument should not be invoked directly. It exists so that the function can properly iterate through object positions by calling itself with deeper positions of the object)

Arguments
object (Object) The object to "randomise""
Returns
(Object) The randomised object
(Note that this is a separate object to the one passed to the function, which remains unchanged)

Peforms a similar function to randomiseStructure(), but instead of selecting random items from an object's arrays, it selects items with a given index

copyStructure()

faria.data.copyStructure( object )

Creates a new object that is exactly the same "shape" as the given object, but where no positions have values assigned
(Note that, were you you to look at the source code for this function, you would see that it actually accepts two arguments, however, the second argument should not be invoked directly. It exists so that the function can properly iterate through object positions by calling itself with deeper positions of the object)

Arguments
object (Object) The object to copy
Returns
(Object) The new, empty object

populateArray()

faria.data.populateArray( length, callback )

Essentially a combination of the fill() and map() prototype functions, this creates an array of given length, and fills each position of the array with the output of a given function
This is particularly useful for when the function you want will yield a different result everytime it is called

Arguments
length (Number) The size of the array you want to create

callback (Function) The function used to fill each position in the array
Returns
(Array) The newly created array 
// let's assume we have defined an object (named "HashedObj")
// whose constructor includes a section of code
// that assigns every new instance of the object with a unique key
// we can use populateArray() to easily create an array of these objects
faria.data.populateArray(4, () => new HashedObj());
// => [ {..., key: '#dfh46', ...}, {..., key: '#xsv5g', ...}, ... ]

extend()

faria.data.extend( object, extension )

A more powerful version of assignNodeValue(), this functions attaches an object structure of any shape or complexity to an existing object, integrating it into any pre-existing shape / struture etc. as needed
(Note that this function is intended for manipulating structures, rather than values, and so trying to change an existing position's value will only work if you are changing it to another value. See examples below)

Arguments
object (Object) The object you want to extend. Best practice would be to create this object as a variable outside of the function, but you can pass it directly and the function will return the extended object

extension (Object) The object to be attached. It can be of any shape or size.
Returns
(Object) The function directly changes the passed object, and while it does give a return value, the best way to use this function is to assign your object elsewhere, and call extend() without assigning it to any variable itself. The initial object will be changed. 
var obj = {};

faria.data.extend(obj, {
  one:1,
  two:2,
  mod:{
    three:3,
    mod_2:{
      target:'hello world'
    }
  }
});

obj;
// => { one: 1,
//    two: 2,
//    mod: { three: 3, mod_2 { target: 'hello world' } } }

var obj = {mod: {four: 4}};

faria.data.extend(obj, {
  mod:{
    three:3, mod_2:{
      target:'hello world'
    }
  }
});

obj;
// => { mod: { four: 4, three: 3, mod_2: { target: 'hello world' } } }

// NOTE that the function can change values into other values
var obj = {one: 'one'};

faria.data.extend(obj, {
  one: 1
});

obj;
// => { one: 1 }

// the function cannot change a value into an object, however
var obj = {mod: 'string'};

faria.data.extend(obj, {
  mod: {mod_2: [object Object]}
});

obj;
// => TypeError: cannot set property [whatever your property name is] of undefined

cloneObject()

faria.data.cloneObject( object )

Provides an identical, but entirely separate version of the given object. Editing the returned object will not be mirrored in the old.
(Note that, were you you to look at the source code for this function, you would see that it actually accepts two arguments, however, the second argument should not be invoked directly. It exists so that the function can properly iterate through object positions by calling itself with deeper positions of the object)

Arguments
object (Object) The object to be cloned
Returns
(Object) An object that is identical to that passed as an argument, but that is it's own, completely separate variable. 
var obj = {
  mod: {
    mod_1: 1,
    mod_2: [ 'a', 'b' ],
    mod_3: 3
  }
};

var newObj = faria.data.cloneObject(obj);

newObj.mod.mod_1 = 'one';

console.log(obj);
// => { mod: { mod_1: 1, mod_2: [ 'a', 'b' ], mod_3: 3 } }

console.log(newObj);
// => { mod: { mod_1: 'one', mod_2: [ 'a', 'b' ], mod_3: 3 } }

shallowSort()

faria.data.shallowSort( object, [desc=true] )

Sorts the given object according to its first layer of key / value pairs
(Note that this function should only be used on a shallow object, whose values are all numbers)

Arguments
object (Object) The object to be sorted

desc (Bool) An optional argument to have the function sort in a descending or ascending pattern
Returns
(Array) An array of [key, value] arrays, sorted according to their value attribute, where key is a key from the given object, and value is its associated value.
(Note that this function deliberately returns not only a separate object to that being sorted, but one of a different type. This is because the ordering of an object's keys cannot be relied upon to remain constant) 
var obj = {
  a: 3,
  b: 1,
  c: 7
};

faria.data.shallowSort(obj);
// => [ [ 'c', 7 ], [ 'a', 3 ], [ 'b', 1 ] ]

faria.data.shallowSort(obj, false);
// => [ [ 'b', 1 ], [ 'a', 3 ], [ 'c', 7 ] ]

objFilter()

faria.data.objFilter( object, callback )

Works like the built-in Array.prototype.filter() method, but for an object.
That is, it filters out any key/value pairs in the object that, when passed to a callback, return "false".
(Note that this function creates a new object, rather than returning a modified one.)

Arguments
object (Object) The object to be filtered

callback (Function) The funcition used to determine whether or not a key/value pair should be included in the filtered object.
(Note that this function should return Booleans only)
Returns
(Object) A new object, containing any key/value pairs from the original that would yield a "true" response from the callback 
var obj = {
  one: { id: 'obj01' },
  two: { id: 'obj02' },
  three: { id: 'obj03' }
};

faria.data.objFilter(obj, i => i.id === 'obj02');
// => { two: { id: 'obj02' } }

var obj = {
  one: { name: 'Carol' },
  two: { name: 'Cheryl' },
  three: { name: 'Cristal' },
  four: { name: 'Carina' },
  five: { name: 'Cherlene' }
};

faria.data.objFilter(obj, i => i.name.includes('Ch'));
// => { two: { name: 'Cheryl' }, five: { name: 'Cherlene' } }
